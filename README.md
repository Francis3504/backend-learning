#Backend Development Learning Projects

This repository contains projects and experiments created while learning backend development using Python and FastApi.Each folder focuses on practicing backend concepts such as authentication,RESTful API design, and application structure.

##Projects Included
A. Authentication System 

A practice project focused on user authentication concepts.

Concepts Practiced
 1. User login and registration
 2. Password handling
 3. API endpoints structure
 4. Request and response validation


B. Notes App
A simple backedn application for managing notes

Concepts Practiced
  1. CRUD  operations(Create,Read,Update,Delete)
  2. API routing
  3. Data handling and validation



C. FastAPI Playground
Small scripts and experiments used to understand FastAPI features.

Concepts Practiced.
1. API testing
2. Routing and parameters
3. Request handling
4. Experimenting eith backend concepts


Requirments
Install dependencies

pip install -r requirements.txt

How to Run
Install Pyhton(htpp://www.python.org/)
Navigate to the project folder and run  the FastAPI application:
uvicorn main:app --reload

Then open your browser and search
http://127.0.0.1:800/docs
